## Key Features

*   **Live Broadcast 24/7:** Non-stop streaming with dynamic mixes of Rock, Pop, and Breakcore.
*   **Real-time Track Display:** Automatically shows album art, title, and artist of the current song, updated in real-time.
*   **Responsive Design:** Looks and functions perfectly across all devices, from desktops to smartphones.
*   **Dynamic Background:** An engaging animated background that creates an immersive visual experience.
*   **One-Click Track Copy:** Easily copy current track information to your clipboard with a single click.
*   **Animate On Scroll (AOS):** Smooth content animations as you scroll, enhancing user experience.
*   **Progressive Web App (PWA):** Install the website as an app on your mobile devices for quick and seamless access.
*   **Broadcast Schedule:** Dedicated sections for day and night programming.

## Technologies Used

The website is built using modern web technologies:

*   **HTML5:** For the page structure and content.
*   **CSS3:** For styling, animations, and responsive design.
*   **JavaScript (ES6+):** For interactivity, player controls, real-time track data fetching, and PWA functionality.
*   **Google Fonts:** For stylish typography.
*   **AOS (Animate On Scroll):** For scroll-triggered animations.

## Development Setup (for contributors)

If you wish to contribute to the website's development or run it locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Nik3310/NikRadofemPlayer-Site.git
    ```
    (Please replace `Nik3310/NikRadofemPlayer-Site.git` with the actual path to your repository if it differs.)
2.  **Open in your IDE (e.g., WebStorm):**
    Launch WebStorm and open the `NikRadofemPlayer-Site` folder.
3.  **View `index.html`:**
    You can open `index.html` directly in your browser or use your IDE's "Open in Browser" feature.
